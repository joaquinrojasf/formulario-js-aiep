# Formulario Web con JavaScript

Este proyecto corresponde a la actividad formativa de la semana 2 del curso Taller de Plataformas Web.

## Funcionalidades

- Formulario con campos:
  - Nombre
  - Correo
  - Mensaje
- Validación de datos con JavaScript
- Uso de eventos DOM (submit)
- Simulación de envío con Promesas
- Manejo de errores

## Tecnologías utilizadas

- HTML
- CSS
- JavaScript

## Integrantes

- Joaquin rojas Figueroa


## Descripción

El sistema valida los datos ingresados por el usuario y simula el envío de información a un servidor utilizando una promesa con tiempo de espera de 2 segundos.
